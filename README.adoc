Tig 是什么？
------------
Tig 是一个基于 ncurses 的 Git 文本模式界面.它主要用作 Git 仓库
浏览器,同时也能在提交前按代码块(chunk)粒度暂存改动,并可作为
各种 Git 命令输出的分页器(pager)使用.

问题反馈与功能建议
------------------
Bug 和功能建议可以通过
https://github.com/jonas/tig/issues[issue 跟踪器] 提交,
也可以发邮件到 https://lore.kernel.org/git/[Git 邮件列表],
注意邮件主题中需要包含 "tig" 一词.其他与 Tig 相关的问题
请使用 Stack Overflow:https://stackoverflow.com/questions/tagged/tig[].

如果是提交 Bug 报告,请附上以下信息:

- 你使用的 Tig 和 ncurses 版本是多少？
  (`tig -v`)
- 你的系统是什么？
  (`uname -a`,`lsb_release -a`)
- 你使用的 Git 版本是多少？
  (`git -v`)
